Yama Puzzle — حزمة Electron (عرض توضيحي)
=====================================

ملفات المشروع:
- package.json
- main.js
- preload.js
- /app/index.html  (اللعبة)
- /assets/icon.png
- /app/audio/forest_loop.wav  (موسيقى خلفية هادئة - بدون حقوق)

طريقة التشغيل (مطلوب Node.js):
1. فك الضغط عن الحزمة.
2. افتح الطرفية في مجلد المشروع.
3. نفّذ: npm install
4. نفّذ: npm run start
   - سيشغل تطبيق Electron للاختبار.

لبناء نسخة قابلة للتشغيل المحمولة (.exe لويندوز):
- نفّذ: npm run dist
- ملاحظة: بناء نسخة ويندوز على نظام غير ويندوز قد يتطلب أدوات إضافية (مثل wine) أو استخدام GitHub Actions.

خدمة GitHub Actions مضافة لملفات البناء التلقائي؛ بعد رفع المشروع إلى GitHub وتفعيل Actions سيرفَع الملف التنفيذي في قسم Releases تلقائيًا.

لو رغبت، أقدر:
- أعدّل نصوص اللعبة أو أضيف مستويات،
- أجهز المستودع على GitHub،
- أو أشرح خطوات رفع وبناء الـ EXE خطوة بخطوة.
